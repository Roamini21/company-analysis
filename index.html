왼쪽에서 기업을 선택하세요

등록된 모든 기업 분석이 왼쪽 목록에 탭처럼 정리됩니다. 항목을 클릭하면 이 영역에 해당 분석이 그대로 펼쳐집니다.

Empty

아직 등록된 기업이 없습니다

분석 HTML 파일을 저장소의 companies 폴더에 올리면 자동으로 여기에 나타납니다.

1
GitHub 저장소에서 companies 폴더로 이동합니다.
2
Add file → Upload files 를 눌러 Claude가 만든 HTML 파일을 끌어다 놓습니다.
3
Commit changes 후 이 페이지를 새로고침하면 새 탭이 생성됩니다.
Setup

호스팅 설정이 필요합니다

이 대시보드는 GitHub Pages 위에서 동작할 때 폴더의 파일을 자동으로 읽어옵니다. 지금은 저장소 정보를 감지하지 못했습니다 (로컬 미리보기이거나 커스텀 도메인일 수 있습니다).

파일 상단의 CONFIG 에서 user 와 repo 를 직접 입력하면 어디서든 동작합니다.

1
GitHub에 company-analysis 같은 이름으로 공개(public) 저장소를 만듭니다.
2
이 index.html 을 저장소 최상위에 올리고, companies 폴더에 분석 파일들을 올립니다.
3
Settings → Pages → Deploy from a branch → main / root 로 게시합니다.
4
https://[아이디].github.io/company-analysis/ 로 접속합니다.
